# Prototype Build Order

## Objective

Get to one polished playable jungle slice as fast as possible.

## Order of work

### 1. Base project sanity

- Open project in UE 5.7
- Save the project cleanly
- Confirm default map settings

### 2. Traversal foundation

- Choose first-person or close third-person camera
- Set up movement tuning
- Validate sprint, crouch, jump, and climb proxies

### 3. Environment blockout

- Rough in terrain path
- Add river or waterfall landmark
- Add ruin destination silhouette

### 4. Atmosphere pass

- Exponential height fog
- Rain system
- Lighting mood for humid jungle conditions
- Ambient jungle audio bed

### 5. Threat beat

- Trigger zone
- Predator cue or stealth pressure
- Safe recovery pocket afterward

### 6. Discovery beat

- Ruin chamber or overlook reveal
- Story prop placement
- End scene framed for screenshots

## Definition of done

- The slice runs from entry to reveal without dead ends.
- The player experiences at least one tension spike and one strong beauty beat.
- The map produces at least three marketable screenshots.
