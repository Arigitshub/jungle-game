# Production Roadmap

## Phase 1. Public presence

- Launch repo
- Publish landing page
- Set baseline SEO and metadata

## Phase 2. Pre-production

- Lock core pillars
- Build vertical slice plan
- Choose engine and technical stack
- Gather visual references

## Phase 3. Prototype

- Create traversal controller
- Build one environment chunk
- Add weather transition and threat encounter
- Record first trailer footage

## Phase 4. Public dev cycle

- Publish regular progress posts
- Improve site with screenshots and clips
- Collect early audience interest

## Phase 5. Expansion

- Add additional biomes
- Add deeper survival systems
- Add stronger narrative structure
