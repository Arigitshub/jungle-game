# JungleGame Unreal Project

This is the Unreal Engine 5.7 project root for Jungle Game.

## Intended first slice

- `Content/Maps/Jungle_Prototype`
- `Content/Blueprints/`
- `Content/Environment/`

## Goal

Build one polished playable environment chunk that proves:

- traversal
- weather mood
- jungle atmosphere
- one stealth or threat beat
- one ruin reveal
